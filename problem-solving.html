Code Readability

https://www.youtube.com/watch?v=SPlS4kW0UbE - line length 45-90 characters, try and write code that doesn't require a comment. Generic names that shouldnt be used in variable/function names as they are ambiguous check, validate, make, get/set, process, start. Try and align with the way that existing code is already written.
